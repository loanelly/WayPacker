## 🇷🇺 Русский

**WayPacker** — умный кроссплатформенный планировщик для путешественников, написанный на JavaFX. Приложение мгновенно генерирует индивидуальный чек-лист вещей на основе 6 ключевых факторов: направления, типа местности, жилья, длительности, сезона и климата.

### 🌟 Ключевые возможности
- ⚡ **Мгновенная генерация:** Получите готовый список вещей в один клик.
- 🧠 **Умный подбор:** Система учитывает 6 важнейших условий вашей поездки.
- 💻 **Кроссплатформенность:** Работает на Windows, macOS и Linux благодаря JavaFX.
- 🎨 **Современный UI:** Чистый, интуитивный и отзывчивый интерфейс.

### 📊 6 ключевых факторов сборки

| Фактор | Описание |
| :--- | :--- |
| **📍 Направление** | Специфика конкретной страны или города |
| **⛰️ Тип местности** | Горы, пляж, мегаполис или дикий лес |
| **🏨 Жилье** | Отель, палатка, хостел или в гостях |
| **⏱️ Длительность** | Поездка на выходные, средний срок или долгий отпуск |
| **🍂 Сезон** | Особенности вещей для зимы, весны, лета или осени |
| **☀️ Климат** | Жара, тропики, затяжные дожди или сильный мороз |
