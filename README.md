# WayPacker

**Language:**  <a href="https://github.com/loanelly/WayPacker/blob/main/README_EN.md"><img src="img/Flags/EN1.png" height="21" align="center"/></a> <a href="https://github.com/loanelly/WayPacker/blob/main/README.md"><img src="img/Flags/result.png" height="26" align="center"/></a>


WayPacker — умный кроссплатформенный планировщик для путешественников, написанный на JavaFX. Приложение мгновенно генерирует индивидуальный чек-лист вещей на основе 6 ключевых факторов: направления, типа местности, жилья, длительности, сезона и климата.
