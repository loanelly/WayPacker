## 🇺🇸 English

**WayPacker** is a smart, cross-platform travel planner built with JavaFX. The app instantly generates a personalized packing checklist based on 6 key factors: destination, terrain, accommodation, duration, season, and climate.

### 🌟 Key Features
- ⚡ **Instant Generation:** Get your tailored packing list in one click.
- 🧠 **Smart Algorithms:** Takes into account 6 vital travel conditions.
- 💻 **Cross-Platform:** Works seamlessly on Windows, macOS, and Linux thanks to JavaFX.
- 🎨 **Modern UI:** Clean, intuitive, and user-friendly interface.

### 📊 The 6 Core Factors

| Factor | Description |
| :--- | :--- |
| **📍 Destination** | Country or city specific requirements |
| **⛰️ Terrain** | Mountains, beach, urban city, or wild forest |
| **🏨 Accommodation** | Hotel, tent, hostel, or staying with friends |
| **⏱️ Duration** | Weekend getaway, mid-range trip, or long vacation |
| **🍂 Season** | Winter, spring, summer, or autumn essentials |
| **☀️ Climate** | Hot, tropical, rainy, or freezing conditions |
